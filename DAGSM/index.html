4. Comparisons with baselines

We compare our method with three baselines that adopt three typical 3D representations, including:

  • Dreamwaltz: using single NeRF representation and SDS loss for optimizing.
  • HumanGaussian: modeling the whole clothed human with a set of 3D-GS and using SD fine-tuned with depth maps as guidance.
  • SO-SMPL: a disentangled method using meshes to represent body and garments and SDS loss as guidance.
